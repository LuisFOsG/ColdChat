# ColdChat

## Requiere
* nodejs

## Instalar Dependencias
```javascript
npm install
```
Con esto creara la carpeta node_modules con los componentes necesarios para correr el programa en la computadora.

## Ejecutar Programa
```javascript
npm run dev
```
Se ejecutara en el LocalHost 3000
